<objective>
Implement metadata fallback search in the WikiJsApi class so that when GraphQL search returns fewer results than the requested limit, a supplementary substring search against page titles and paths fills the gap.

Purpose: Acronym searches (e.g., "COA"), short tokens, and path segments (e.g., "mendix") that Wiki.js's search engine misses due to stemming/tokenization now return matching pages.

Output: Modified `src/api.ts` with `searchPagesByMetadata()` private method wired into `searchPages()`, and comprehensive unit tests in `tests/api.test.ts`.
</objective>
