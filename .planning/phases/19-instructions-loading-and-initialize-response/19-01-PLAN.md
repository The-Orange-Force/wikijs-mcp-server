       - On error (catch): `console.warn("Could not load instructions from {path}: {error.message}. Using default instructions.")`, return `DEFAULT_INSTRUCTIONS`

    - Test: no path -> returns DEFAULT_INSTRUCTIONS, no console output
    - Test: valid path -> calls readFile, returns content, logs info
    - Test: readFile throws ENOENT -> returns DEFAULT_INSTRUCTIONS, logs warning with path and error
    - Test: readFile throws EACCES -> returns DEFAULT_INSTRUCTIONS, logs warning with path and error
    - Test: DEFAULT_INSTRUCTIONS contains all 5 topics
    - Test: DEFAULT_INSTRUCTIONS does NOT contain tool names (search_pages, get_page, list_pages)
