Create `src/mcp-tools.ts` that exports a `createMcpServer(wikiJsApi: WikiJsApi): McpServer` factory function.

**Structure:**
1. Import McpServer from `@modelcontextprotocol/sdk/server/mcp.js`
2. Import `z` from `zod`
3. Import `WikiJsApi` from `./api.js`

4. Create the factory function that:
   a. Creates `new McpServer({ name: "wikijs-mcp", version: "1.3.0" })`
   b. Registers all 17 tools using `mcpServer.registerTool()`

**Critical rules for registerTool():**
- `inputSchema` must be a FLAT object of Zod types, NOT wrapped in z.object(). Example: `{ id: z.number().int().positive() }` not `z.object({ id: ... })`
- Do NOT import and pass the Zod schemas from schemas.ts directly (they are z.object wrappers). Instead, extract the shape properties or define them inline.
- Handler functions must return `{ content: [{ type: "text", text: JSON.stringify(result, null, 2) }] }` for success
- Handler functions must return `{ content: [{ type: "text", text: "Error: ..." }], isError: true }` for errors
- Wrap every handler body in try/catch

**All 17 tools to register (use WikiJsApi method names from the interfaces block above):**

Page tools (10):
- `get_page` -- inputSchema: `{ id: z.number().int().positive().describe("Page ID") }` -- calls `wikiJsApi.getPageById(id)`
- `get_page_content` -- inputSchema: `{ id: z.number().int().positive().describe("Page ID") }` -- calls `wikiJsApi.getPageContent(id)`

- `search_pages` -- inputSchema: `{ query: z.string().min(1).describe("Search query"), limit: z.number().int().positive().optional().describe("Max results") }` -- calls `wikiJsApi.searchPages(query, limit)`
- `create_page` -- inputSchema: `{ title: z.string().min(1).describe("Page title"), content: z.string().describe("Markdown content"), path: z.string().min(1).describe("Page path e.g. folder/page"), description: z.string().optional().describe("Page description") }` -- calls `wikiJsApi.createPage(title, content, path, description)`
