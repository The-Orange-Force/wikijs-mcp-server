<objective>
Implement RFC 9728 Protected Resource Metadata endpoint and scope-to-tool mapping for the WikiJS MCP server.

Purpose: MCP clients need to discover this server's authorization requirements (which Azure AD tenant to authenticate against, what scopes are available) via a standardized endpoint. The scope-to-tool mapping defined here becomes the single source of truth that Phase 4/5 auth middleware will consume for enforcement.

Output: `src/scopes.ts` (scope mapping module), updated `src/server.ts` (with `buildApp()` factory and `/.well-known/oauth-protected-resource` route), updated `src/config.ts` (optional `MCP_RESOURCE_DOCS_URL`), integration and unit tests.
</objective>
